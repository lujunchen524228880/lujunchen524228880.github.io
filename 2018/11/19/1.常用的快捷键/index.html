初识python

Created at 2018-11-19 Updated at 2019-04-15 Category 语言基础 Tag Python / PEP8

1.常用的快捷键
ctrl + s -保存
ctrl + / -注释和取消注释
ctrl + b -sublime中的编译执行
ctrl + c -复制，拷贝
ctrl + v -粘贴
ctrl + a -全选
ctrl + z -撤销
ctrl + shift + z -反撤销
ctrl + x -剪切
ctrl + f -查找
按住shift不放点鼠标 - 选中指定范围
按住ctrl不放点鼠标 - 多光标操作
print是python中的内置函数，功能是在控制台输出括号里面的内容

print(‘hello world,你好世界！’)
2.注释
注释就是代码中不参与执行的说明文字
当行注释：在文字前加#
“””
多行注释：用三个单引号或者三个双引号引起来（一般使用双引号）
多行注释第一行
多行注释第二行
“””

3.标志符
程序中专门用来命名的符号叫做标志符
python中标志符的要求：所有的标志符是由字母，数字和下划线组成，并且数字不能开头
在python3.x中标志符还可以包括中文

  1. 变量名 = 值

num = 10
数字 = 10 #python3.x中标志符可以使用中文
5.缩进
一行代码前的空格叫缩进，python中每行代码前缩进的个数是有规范的
不能随便缩进，也不能在该缩进的时候不缩进

num = 10             #IndentationError缩进错误

6.常用数据类型
int - 整型（整数）,例如：10，-100,123,0，+321 float - 浮点型(小数),例如：3.141516927
str - 字符串， 例如：‘今天天气真好！’,“hello!”
bool - 布尔,值只有True和False两个，True - 真成立，False - 假不成立
list(列表)、dict(字典)、set(集合)、tuple(元祖)、迭代器、生成器、function等。

7.两个常用的内置函数 ：
print(),input()
a.输出函数：print() - 在控制台输出括号里面的内容
print(变量)，print(数据),print(数据1，数据2)

num = 100
int2 = 1000
message = ‘你好吗’
print(num, int2, message, 122) # 同时输出多个数据
注意：一个print打印的数据占一行，如果一个print同时打印 多个数据，多个数据之间显示的时候是用空格隔开的

b.输入函数：input() - 从输入设备上获取输入的内容到程序中
输入结果 = input(提示信息)

result = input(‘请输入账号：’)
print(‘结果：’,result)
注意：当程序执行到input函数的时候，程序会停下来等待输入完成

写程序用来收集一个人的姓名，年龄，并且打印出来

name = input(“请输入你的名字:”)
age = input(‘请输入你的年龄:’)
print(name,age)
8.变量
变量是用来在内存中开辟空间保存数据的
怎么声明/定义变量
语法：变量名 = 数据 #
说明：a.变量名 - 标志符，并且不能是关键字
见名知义（看到变量名大概能知道这个变量中存的数据）
所有字母都小写，如果变量名是由多个单词组成，单词之间用下划线隔开
b.= - 赋值符号 ，将赋值符号右边的数据赋值到左边的变量
c.数据 - 数据可以是不同类型的值，也可以是已经声明过的变量或者是运算表达式

student_name = ‘xiaoming’
num2 = 12 + 23
print(num2)
已经声明过得变量可以重新赋值，新赋的值会覆盖原来的值

num = 12
num = 13
num = ‘xiaohua’
print(num)
9.同时声明多个变量

a = b = c = 100 # 同时声明多个变量赋相同的值
print(a,b,c)
a1,a2,a3 = 10,20,30 # 同时声明多个变量赋不同的值
print(a1,a2,a3)
python中声明变量和给变量赋值的基本原理：
先开辟空间将被数据存起来，数据需要多大的内存空间就开辟多大的内存。然后再将变量指向对应的内存区域

单位：
8位（bit） = 1字节（byte）
1024字节 = 1kb
1024kb = 1Mb
1024Mb = 1Gb
1024Gb = 1T
10.数学运算符
python支持的运算符有：数学运算符，比较运算符，逻辑运算符，赋值运算符
a.数学运算符：+(加)，- （减），* （乘），/ （除）, // （整除）, %（取余）, **（幂运算）
前面的加减乘除和数学中的加减乘除的功能一样

print(2+33)
print(4-3)
print(4*3)
print(6/4)
//求商结果只保留整数部分

print(9//5) #1
print(9.2//5) #1.0
% - 求余数

print(9%5) #4
print(9.2%5) #4.19999999999
- 幂运算（XY 求x的y次方）

print(2**8) #256

print(19 + 23 * 2 - 10 / 5) #63.0
括号可以改变运算顺序
